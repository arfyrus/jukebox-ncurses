# Jukebox Player

Simple jukebox simulation using Rust and ncurses package.

## Controls

| Key              | Action                                            |
|------------------|---------------------------------------------------|
| <kbd>q</kdb>     | Quits the program                                 |
| <kbd>k,j</kbd>   | Moves up and down respectively                    |
| <kbd>Enter</kbd> | Switches highlighted item from song list to queue |
